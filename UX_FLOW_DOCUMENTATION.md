## 用户旅程图 (User Journey Map)

### 完整用户旅程：从进入应用到完成一次复制交易

```
┌─────────────────────────────────────────────────────────────────┐
│                     用户旅程：完成复制交易                        │
└─────────────────────────────────────────────────────────────────┘

阶段 1: 进入应用
├─ 用户访问网站首页
├─ 自动跳转至登录页面 (/login)
├─ 用户可选择：
│  ├─ 使用邮箱登录
│  ├─ 使用社交账号登录 (Telegram/Phantom/MetaMask)
│  └─ 扫描QR码登录（移动端）
└─ 点击"Sign Up"跳转注册页面 (/signup)

阶段 2: 注册/登录
├─ 注册流程：
│  ├─ 输入邮箱
│  ├─ 输入邀请码（可选）
│  ├─ 点击"Sign Up"按钮
│  └─ 或使用社交账号注册
└─ 登录成功 → 跳转至主页面 (/)

阶段 3: 浏览钱包排名
├─ 进入 CopyTrade 仪表板
├─ 查看钱包排名表（默认显示Rank视图）
├─ 可执行操作：
│  ├─ 使用搜索框搜索钱包名称
│  ├─ 选择时间范围（1D/7D/30D）
│  ├─ 切换视图（Rank/Radar/CopyTrade/SnipeX）
│  ├─ 选择分类过滤器（All/Launchpad SM/Smart Money等）
│  └─ 浏览钱包数据（PnL、Win Rate、交易数等）
└─ 找到感兴趣的钱包

阶段 4: 创建复制交易
├─ 点击钱包行中的"Copy"按钮
├─ 跳转至创建复制交易页面 (/create-copy-trade)
├─ 填写表单：
│  ├─ 钱包地址（自动填充）
│  ├─ 选择买入方法（Max Buy Amount/Fixed Buy/Fixed Ratio）
│  ├─ 选择金额（10/25/50/100 SOL）
│  ├─ 选择卖出方法（Copy Sell/Not Sell/TP & SL/Adv Strategy）
│  ├─ 配置高级设置（可选）
│  └─ 选择预设（可选）
└─ 点击"Confirm"确认

阶段 5: 确认与完成
├─ 系统验证配置
├─ 显示确认信息
├─ 返回主页面
└─ 复制交易开始运行

```

## 交互流程说明 (Interaction Flow Descriptions)

### 流程 1: 用户登录

```
步骤 1: 用户访问应用
├─ 系统检测未登录状态
└─ 自动重定向至 /login

步骤 2: 用户选择登录方式
├─ 选项 A: 邮箱登录
│  ├─ 输入邮箱地址
│  ├─ 点击"Log In"按钮
│  └─ 系统验证并登录
│
├─ 选项 B: 社交登录
│  ├─ 点击社交平台按钮（Telegram/Phantom/MetaMask）
│  ├─ 弹出第三方登录窗口
│  └─ 授权后返回并登录
│
└─ 选项 C: QR码登录
   ├─ 使用移动端App扫描QR码
   └─ 确认后登录

步骤 3: 登录成功
└─ 跳转至主页面 (/)
```

### 流程 2: 注册新用户

```
步骤 1: 用户点击"Sign Up"
└─ 跳转至 /signup

步骤 2: 填写注册信息
├─ 输入邮箱（必填）
├─ 输入邀请码（可选）
│  └─ 提示："邀请码绑定后无法更改"
└─ 点击"Sign Up"按钮

步骤 3: 注册成功
└─ 自动登录并跳转至主页面 (/)
```

### 流程 3: 浏览钱包并创建复制交易

```
步骤 1: 进入CopyTrade仪表板
├─ 默认显示Rank视图
├─ 显示钱包排名表（按PnL排序）
└─ 展示多个指标列

步骤 2: 筛选和搜索钱包
├─ 使用搜索框输入钱包名称
├─ 选择时间范围（1D/7D/30D）
├─ 切换视图（Rank/Radar/CopyTrade/SnipeX）
└─ 选择分类过滤器（All/Launchpad SM等）

步骤 3: 选择目标钱包
├─ 浏览表格数据
├─ 查看PnL、Win Rate、交易数等指标
└─ 找到感兴趣的钱包

步骤 4: 点击"Copy"按钮
└─ 跳转至 /create-copy-trade，并传递钱包信息

步骤 5: 配置复制交易
├─ 钱包地址自动填充
├─ 选择买入方法：
│  ├─ Max Buy Amount（最大买入金额）
│  ├─ Fixed Buy（固定买入）
│  └─ Fixed Ratio（固定比例）
├─ 选择金额：10/25/50/100 SOL
├─ 选择卖出方法：
│  ├─ Copy Sell（复制卖出）
│  ├─ Not Sell（不卖出）
│  ├─ TP & SL（止盈止损）
│  └─ Adv Strategy（高级策略）
├─ 配置高级设置（可选）：
│  ├─ 展开高级设置
│  ├─ 设置滑点参数
│  └─ 其他高级选项
└─ 选择预设（如果有）

步骤 6: 确认创建
├─ 点击"Confirm"按钮
├─ 系统验证配置
└─ 返回主页面，复制交易开始运行
```

### 流程 4: 查看钱包总览

```
步骤 1: 进入钱包总览页面
└─ 点击导航栏"Portfolio"或访问 /wallet

步骤 2: 选择时间范围
└─ 点击时间范围按钮（1D/7D/30D/All）

步骤 3: 查看统计信息
├─ 查看总余额和涨跌幅
├─ 查看SOL余额和USD等值
├─ 查看胜率和交易数
└─ 查看总盈亏

步骤 4: 浏览交易历史
├─ 查看交易列表
├─ 查看每笔交易的详细信息
│  ├─ 交易类型（买入/卖出）
│  ├─ 币种
│  ├─ 数量
│  ├─ 价格
│  ├─ 时间
│  └─ 状态
└─ 可滚动查看更多交易
```

### 流程 5: 查看市场数据

```
步骤 1: 进入市场数据页面
└─ 访问 /market（或通过导航）

步骤 2: 选择分类
└─ 点击分类标签（All/Trending/Top Gainers等）

步骤 3: 浏览市场数据
├─ 查看币种列表
├─ 查看价格和涨跌幅
├─ 查看24h成交量
└─ 查看市值

步骤 4: 点击币种（可选）
└─ 跳转至币种详情页面（如果实现）
```

## 关键交互元素标注 (Key Interaction Elements)

### 1. 导航栏交互元素

| 元素 | 位置 | 交互行为 | 状态反馈 |
|------|------|----------|----------|
| 导航项 | 顶部导航栏 | 点击跳转 | 当前页高亮（绿色背景+文字） |
| 搜索框 | 顶部右侧 | 输入搜索 | 实时过滤结果 |
| "Create Copy Trade"按钮 | 顶部右侧 | 点击跳转创建页 | 悬停时颜色变亮 |
| 通知图标 | 顶部右侧 | 点击显示通知 | 红点提示未读 |
| 用户图标 | 顶部右侧 | 点击显示菜单 | 悬停时背景变亮 |

### 2. 侧边栏交互元素

| 元素 | 位置 | 交互行为 | 状态反馈 |
|------|------|----------|----------|
| Wallet/Track/Monitor/Renames | 左侧边栏 | 点击切换视图 | 当前项高亮（绿色背景+文字） |
| Groups项 | 左侧边栏 | 点击筛选 | 选中状态显示 |
| "+Groups"按钮 | 左侧边栏 | 点击创建新组 | 悬停时背景变亮 |

### 3. CopyTrade表格交互元素

| 元素 | 位置 | 交互行为 | 状态反馈 |
|------|------|----------|----------|
| 搜索框 | 顶部 | 输入搜索 | 实时过滤表格 |
| 时间范围按钮（1D/7D/30D） | 顶部 | 点击切换时间范围 | 选中按钮高亮 |
| 视图切换（Grid/List） | 顶部 | 点击切换视图 | 当前视图高亮 |
| 主标签（Rank/Radar等） | 顶部 | 点击切换视图 | 选中标签绿色背景+白字 |
| 分类过滤器 | 顶部 | 点击筛选 | 选中项绿色背景+文字 |
| 表格行 | 表格主体 | 悬停 | 行背景变亮 |
| "Copy"按钮 | 每行末尾 | 点击创建复制交易 | 悬停时颜色变亮，点击跳转 |

### 4. 创建复制交易表单交互元素

| 元素 | 位置 | 交互行为 | 状态反馈 |

## 响应式设计说明

### 桌面端（≥1024px）
- 显示完整导航栏和侧边栏
- 表格显示所有列
- 多列布局

### 平板端（768px - 1023px）
- 隐藏部分导航项或使用下拉菜单
- 侧边栏可折叠
- 表格可能需要横向滚动

### 移动端（<768px）
- 导航栏简化为图标和关键按钮
- 侧边栏隐藏，使用抽屉式菜单
- 表格横向滚动
- 表单使用单列布局
- 按钮和输入框尺寸适当增大
